# 🌟 Функціонал сайту TripsForUA

## 🏠 Публічна частина

### Головна сторінка (/)
- ✨ Hero секція з призивом до дії
- 📊 Статистика компанії (тури, клієнти, відгуки)
- 🗺️ Популярні напрямки (карточки з фото)
- 📅 Найближчі тури (календар)
- ℹ️ Секція "Чому ми?" з перевагами
- 📞 Контактна форма

### Сторінка напрямків (/destinations)
- 🌍 Список всіх країн
- 🔍 Фільтр по континентах (Європа, Африка, Азія)
- 🖼️ Красиві карточки з фото
- 🏴 Прапори країн (emoji)
- 📝 Короткий опис кожного напрямку

### Детальна сторінка напрямку (/destinations/:slug)
- 📷 Hero зображення
- 📖 Повний опис країни
- 🖼️ Фотогалерея
- 📋 Список доступних турів до цього напрямку
- 🔗 Переходи на тури

### Детальна сторінка туру (/tours/:id)
- 🏔️ Великий hero банер
- 📝 Повний опис туру
- ⭐ Основні моменти (highlights)
- ✅ Що включено в ціну
- ❌ Що не включено
- 📅 Програма по днях (itinerary)
- 💰 Калькулятор ціни
- 📝 Форма бронювання
- 👥 Показ вільних місць

### Форма бронювання
- 📝 Ім'я, email, телефон
- 👥 Кількість осіб
- 💬 Коментар (опціонально)
- 💰 Автоматичний розрахунок загальної суми
- ✉️ Підтвердження через toast повідомлення

## 📊 API Endpoints

### Публічні:
- `GET /api/destinations` - Список напрямків
- `GET /api/destinations/:slug` - Деталі напрямку
- `GET /api/tours` - Список турів (з фільтрами)
- `GET /api/tours/:id` - Деталі туру
- `POST /api/bookings` - Створити бронювання
- `POST /api/auth/login` - Авторизація

### Адмін (з JWT):
- `POST /api/tours` - Створити тур
- `PUT /api/tours/:id` - Оновити тур
- `DELETE /api/tours/:id` - Видалити тур
- `POST /api/destinations` - Створити напрямок
- `PUT /api/destinations/:id` - Оновити напрямок
- `DELETE /api/destinations/:id` - Видалити напрямок
- `GET /api/bookings` - Список бронювань
- `PUT /api/bookings/:id` - Оновити бронювання
- `GET /api/auth/me` - Поточний користувач
